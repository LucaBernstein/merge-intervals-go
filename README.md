# Merge Intervals

This package merges overlapping number intervals.

## Usage

```bash
go run . <input intervals>
```

## Example

The input `` should yield the output ``.

```bash
# Input
go run . "[25,30] [2,19] [14,23] [4,8]"

# Output
[2,23] [25,30]
```
